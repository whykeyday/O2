BGM Placeholder Files
=====================

Place your background music files here with the following names:

1. bgm_main.mp3     - Main menu background music
2. bgm_route1.mp3   - Bicycle Path route music
3. bgm_route2.mp3   - Fire Path route music
4. bgm_route3.mp3   - Human Path route music
5. bgm_route4.mp3   - Sun Path route music

After adding the files:
- Select each file in Unity Inspector
- Set Load Type: Streaming
- Set Compression Format: Vorbis
- Set Quality: 0.5-0.7
