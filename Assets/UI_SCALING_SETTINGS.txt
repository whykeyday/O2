🎮 使用说明 HOW TO USE:
===============================================

1. 在Unity Editor中测试
   ---------------------
   - 关闭Unity并重新打开项目
   - 在Game窗口中点击"Free Aspect"或设置分辨率
   - 调整Game窗口大小，UI会自动缩放

2. Build游戏后
   ---------------------
   - Build游戏后启动EXE文件
   - 窗口边缘可以拖动调整大小
   - UI会根据窗口大小自动缩放
   - 保持2560x1440的视觉比例

===============================================
📐 为什么使用2560x1440参考分辨率？
===============================================

参考分辨率越大 = UI显示越小

例如：
- 640x360   → UI非常大（适合移动设备）
- 1280x720  → UI适中
- 1920x1080 → UI较小
- 2560x1440 → UI很小（适合大显示器）

在22寸1920x1080显示器上：
- 参考2560x1440的UI会缩小到约75%大小
- 这样可以在不滚动的情况下看到完整内容

===============================================
🔧 如果UI大小还不合适：
===============================================

方法1：修改参考分辨率
   UI太大 → 提高分辨率 (如3840x2160)
   UI太小 → 降低分辨率 (如1920x1080)

方法2：调整Match值
   Match = 0   → 完全适应宽度
   Match = 0.5 → 平衡（推荐）
   Match = 1   → 完全适应高度

方法3：在Unity中手动调整
   - 选择Canvas对象
   - 找到Canvas Scaler组件
   - 调整Reference Resolution和Match值
   - 实时查看效果
